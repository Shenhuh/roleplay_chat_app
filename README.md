# AI Roleplay Chat Platform

A community-driven AI roleplay chat application where users can chat with AI-powered characters from any game or fictional universe. Built with Next.js, Supabase, and DeepSeek AI.

## Features

- **AI-Powered Chat** — Roleplay conversations with characters powered by DeepSeek, with personalities driven by structured lore data.
- **Mood System** — Characters track a dynamic mood score that shifts based on conversation tone, affecting how they respond over time.
- **Relationship Graph** — Visual node-based system (built with React Flow) to map and manage relationships between characters.
- **Community Contributions** — Anyone can suggest edits to character lore, nicknames, and other details, or propose new relationships between characters.
- **Voting System** — Community members can upvote/downvote submitted contributions to help surface the most accurate information.
- **Admin Panel** — Full CRUD management for characters, factions, regions, monsters, and world lore, plus a review dashboard to approve or reject community submissions.
- **Game-Agnostic Design** — Character data structure isn't tied to any single game, allowing the platform to support multiple fictional universes.

## Tech Stack

- **Frontend:** Next.js (App Router), React, Tailwind CSS, shadcn/ui
- **Backend:** Supabase (PostgreSQL, Storage)
- **AI:** DeepSeek API
- **Visualization:** React Flow (relationship graphs)
